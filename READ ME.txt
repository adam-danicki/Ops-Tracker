How to start the app:
 - docker compose up
 - docker compose up --build (if not built before)

To open docs:
 - http://localhost:8000/docs

To inspect DB in docker:
 - docker exec -it ops_tracker_db psql -U ops -d ops_tracker
Then inside psql:
 - \dt          = list tables
 - \d projects  = describe a table
 - \q quit      = quit
